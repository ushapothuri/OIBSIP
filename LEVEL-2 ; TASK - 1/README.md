# ✨ Nova Calc - Glassmorphism Calculator

A modern, interactive calculator built with **HTML, CSS, and JavaScript** featuring a futuristic **glassmorphism UI**, smooth animations, theme switching, calculation history, and responsive design.

Nova Calc focuses on delivering a premium user experience with a clean interface inspired by modern mobile applications.

---

## 🚀 Features

### 🧮 Smart Calculator

- Basic arithmetic operations:
  - Addition (+)
  - Subtraction (−)
  - Multiplication (×)
  - Division (÷)
  - Percentage (%)

- Real-time calculation display
- Decimal number support
- Clear and delete functionality
- Error handling for invalid operations

---

## 🎨 Glassmorphism UI Design

Nova Calc includes a modern glass-inspired interface:

- Transparent glass cards
- Blur effects
- Gradient backgrounds
- Floating UI elements
- Smooth hover animations
- Premium button interactions

Designed with a futuristic **Nova-inspired visual style**.

---

## 🌗 Dark & Light Theme

The application supports dynamic themes:

### 🌙 Dark Mode

- Neon gradients
- Purple and cyan highlights
- Futuristic calculator appearance


### ☀️ Light Mode

- Bright glass interface
- Soft gradients
- High contrast buttons
- Clean modern appearance


Theme can be switched instantly using the theme toggle button.

---

## 📜 Calculation History

The calculator includes a slide-out history panel.

Features:

- History hidden by default
- Opens using the menu button
- Smooth drawer animation
- Background blur overlay
- Close using:
  - Close button
  - Outside click

---

## 📱 Responsive Design

Nova Calc works smoothly across different devices:

- Desktop
- Tablets
- Mobile screens

Responsive features:

- Adaptive calculator size
- Mobile-friendly buttons
- Optimized layouts
- Touch-friendly interactions

## ⌨️ Keyboard Support

Users can interact using their keyboard:

Supported keys:
0-9 → Numbers

/ → Operations

Enter → Calculate

Backspace → Delete
