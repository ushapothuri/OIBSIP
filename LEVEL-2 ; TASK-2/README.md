# Emma Watson Tribute Page

## Level 2 - Task 2

This project is created as part of **Level 2 - Task 2**, which focuses on designing and developing a **Tribute Page** using HTML and CSS.

The webpage is dedicated to **Emma Watson**, showcasing her journey as an actress, activist, and inspiration. The design focuses on creating an elegant, modern, and responsive single-page tribute website.

---

## 🌟 Project Overview

The Emma Watson Tribute Page highlights:

- Her early life and background
- Career achievements
- Famous movie roles
- Activism and social contributions
- Inspirational quotes

The website uses a premium dark theme with glassmorphism elements to provide a visually appealing user experience.

---

## 🚀 Features

- Responsive single-page design
- Modern navigation bar
- Full-screen hero section with image overlay
- About section
- Career highlight cards
- Activism section with statistics
- Inspirational quote section
- Glassmorphism UI design
- Smooth scrolling navigation
- Hover animations
