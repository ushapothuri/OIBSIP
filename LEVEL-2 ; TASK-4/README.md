# Login Authentication System

## Project Overview

A simple and professional Login Authentication System developed using HTML, CSS, and JavaScript.
The application allows users to register, login, and access a personal dashboard with authentication-based access.

## Features
- User Registration
- User Login Authentication
- Form Validation
- Password Strength Indicator
- Password Visibility Toggle
- Light/Dark Mode Toggle
- User Dashboard
- Display User Details
- Session Management
- Logout Functionality
- Responsive Design

## Technologies Used
- HTML5
- CSS3
- JavaScript
- LocalStorage

## Working
- User details are stored using browser LocalStorage.
- During login, entered credentials are verified.
- Successful login creates a user session.
- The dashboard displays user information.
- Logout removes the active session.

## Purpose
This project was developed as an internship task to demonstrate frontend development skills, JavaScript logic, authentication flow, and user interface design.
