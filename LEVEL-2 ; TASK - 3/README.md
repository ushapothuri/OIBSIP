TaskFlow - To-Do Web Application
Project Overview

TaskFlow is a simple and modern To-Do web application that helps users manage their daily tasks efficiently. Users can add, edit, delete, complete, and organize tasks with a clean and responsive interface.

The project is developed as an internship-level task using HTML, CSS, and JavaScript with browser local storage support.

Features
Add new tasks
View pending tasks
Move tasks between pending and completed sections
Mark tasks as completed
Edit existing tasks
Delete tasks
Display task creation date and completion date
Task statistics:
Total tasks
Pending tasks
Completed tasks
Light and Dark mode toggle
Data persistence using Local Storage
Responsive modern UI design
Technologies Used
HTML5
Used for creating the structure of the application.
CSS3
Used for styling, responsive layout, animations, and dark mode design.
JavaScript (ES6)
Used for task management functionality and local storage handling.
